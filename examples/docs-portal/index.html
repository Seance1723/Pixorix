A structure built for teams, not toy examples

SCSS layers

Tokens, base, layout, utilities, components, advanced components, and themes.

JS layers

Core app runtime, registry, DOM/events/state helpers, theme hooks, and motion presets.

Theme API

Runtime switching with semantic CSS variables

<html data-px-theme="dark" data-px-brand="default">

const app = initPixorix(document);
app.getThemeController().setTheme('light');
